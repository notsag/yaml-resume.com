yaml-resume

The open source initiative to create a YAML-based standard for resumes and tools to handle them.

What is yaml-resume?

It is a Python package that provides:

  • A CLI that will help you create a well-formed resume in YAML and generate html/pdf files from it.
  • A library for processing YAML resume.

Why yaml?

YAML is lightweight, easy and human readable. This means that it is easy to create and understand a resume in this format.

It is also easy to process YAML files with the multiple libraries available.

Check the Get-Started page to learn how to use it or the Schema page to have an example of a resume in YAML.
